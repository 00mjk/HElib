Computes the \char`\"{}product\char`\"{} of two handles. 

The key-\/\+ID\textquotesingle{}s and powers of X must match, else an error state arises, which is represented using a key-\/\+ID of -\/1 and returning false. Also, note that inputs may alias outputs.
