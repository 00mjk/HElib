\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class helib\+::\+Hyper\+Cube$<$ T $>$}

A multi-\/dimensional cube. 

Such an object is initialized with a \mbox{\hyperlink{classhelib_1_1_cube_signature}{Cube\+Signature}}\+: a reference to the signature is stored with the cube, and so the signature must remain alive during the lifetime of the cube, to prevent dangling pointers.
