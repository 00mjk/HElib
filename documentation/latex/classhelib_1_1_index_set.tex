\doxysubsection{Detailed Description}
A dynamic set of non-\/negative integers. 

You can iterate through a set as follows\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{for} (\textcolor{keywordtype}{long} i = s.first(); i <= s.last(); i = s.next(i)) ...}
\DoxyCodeLine{for (\textcolor{keywordtype}{long} i = s.last(); i >= s.first(); i = s.prev(i)) ...}
\end{DoxyCode}
