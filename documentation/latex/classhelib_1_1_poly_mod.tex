\doxysubsection{Detailed Description}
An object that contains an {\ttfamily N\+T\+L\+::\+Z\+ZX} polynomial along with a coefficient modulus {\ttfamily p2r} and a polynomial modulus {\ttfamily G}. 

A {\ttfamily \mbox{\hyperlink{classhelib_1_1_poly_mod}{Poly\+Mod}}} object can be considered to be an element of $\mathbb{Z}_{p^r}[x]/G$ where $p^r$ and $ G$ are the passed-\/in parameters {\ttfamily p2r} and {\ttfamily G}.

This allows for inter-\/{\ttfamily \mbox{\hyperlink{classhelib_1_1_poly_mod}{Poly\+Mod}}} operations with the modulo operations performed automatically.

General usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classhelib_1_1_poly_mod}{helib::PolyMod}} poly(input\_data, p2r, G);}
\end{DoxyCode}
 Calling an operation on a default constructed {\ttfamily \mbox{\hyperlink{classhelib_1_1_poly_mod}{Poly\+Mod}}} will throw an {\ttfamily \mbox{\hyperlink{classhelib_1_1_logic_error}{helib\+::\+Logic\+Error}}}.
