\doxysubsection{Detailed Description}
An implementation of \mbox{\hyperlink{classhelib_1_1_replicate_handler}{Replicate\+Handler}} that explicitly returns all the replicated ciphertexts in one big vector. 

This is useful mostly for debugging purposes, for real parameters it would take a lot of memory.
