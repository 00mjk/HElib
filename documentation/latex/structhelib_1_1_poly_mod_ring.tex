\doxysubsection{Detailed Description}
Lightweight type for describing the structure of a single slot of the plaintext space. 

A single slot of the plaintext space is isomorphic to $\mathbb{Z}[X]/(G(x),p^r)$ for some irreducible factor G of $\Phi_m(X)$, so the main useful members of this {\ttfamily struct} are {\ttfamily p}, {\ttfamily r}, {\ttfamily G}, and {\ttfamily p2r}.

The fields of this {\ttfamily struct} are all {\ttfamily const}, so they should be determined at the time of construction.

\begin{DoxyNote}{Note}
This {\ttfamily struct} aggregates this often-\/useful information into a single placeholder for convenience. 
\end{DoxyNote}
