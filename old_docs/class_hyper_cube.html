Detailed Description

template<class T>
class HyperCube< T >

A multi-dimensional cube.

Such an object is initialzied with a CubeSignature: a reference to the signature is stored with the cube, and so the signature must remain alive during the lifetime of the cube, to prevent dangling pointers.
