Detailed Description

template<class type>
class PlaintextBlockMatrixInterface< type >

A somewhat less abstract interface for linear transformations.

A block matrix implements linear transformation over the base field/ring (e.g., Z_2, Z_3, Z_{2^8}, etc.) The method get(out, i, j) copies the element at row i column j of a matrix into the variable out. The type of out is mat_R (so either mar_GF2 or mat_zz_p. A return value of true means that the entry is zero, and out is not touched.
